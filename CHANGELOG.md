# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-01

### Added

- Initial release of the Islamic Open Finance OpenAPI specification
- OpenAPI 3.1 specification covering all 105 specialized rails across 12 categories
- Authentication schemes including OAuth 2.0, API key, and JWT bearer token
- Webhook event schemas for contract lifecycle, compliance, and ledger events
- Multi-tenant request/response schemas with tenant-scoped headers
- Shariah compliance metadata fields on all Islamic contract endpoints
